# AutoPartsWarehouse

[![.NET Build and Test](https://github.com/Evklidiy/lab5/actions/workflows/main.yml/badge.svg)](https://github.com/Evklidiy/lab5/actions/workflows/main.yml)

**Лабораторная работа №5**
**Тема:** Разработка интерфейса приложения баз данных с использованием аутентификации и авторизации.
**Вариант:** 13 (Предметная область: Система управления складом автозапчастей).

## Описание проекта
Веб-приложение на базе ASP.NET Core MVC для управления складом автозапчастей. Приложение реализует систему учета товаров, поставщиков и поставок с разграничением прав доступа.

## Функционал
*   **Аутентификация и Авторизация:** Система входа/регистрации на базе ASP.NET Core Identity.
*   **Ролевая модель:** Разделение прав доступа (Администратор, Пользователь).
*   **CRUD операции:** Полный цикл управления данными для таблиц:
    *   Запчасти (AutoParts)
    *   Поставщики (Suppliers)
    *   Поставки (Supplies) — связь "один-ко-многим".
*   **Поиск и Фильтрация:** Фильтрация запчастей по названию и производителю.
*   **Пагинация:** Постраничный вывод больших списков данных.
*   **Администрирование:** Панель для управления пользователями и назначения ролей.

## Технологический стек
*   **Платформа:** .NET 8.0 (ASP.NET Core MVC)
*   **ORM:** Entity Framework Core
*   **База данных:** MS SQL Server (MonsterASP)
*   **Тестирование:** xUnit, Moq, EF Core InMemory
*   **CI/CD:** GitHub Actions

## Запуск проекта
1.  Клонировать репозиторий:
    ```bash
    git clone https://github.com/Evklidiy/lab5.git
    ```
2.  Настроить строку подключения в `appsettings.json`.
3.  Применить миграции и запустить:
    ```bash
    dotnet run
    ```
